🗄️ Supabaseって何？

データを保存する「クラウド上の倉庫」。

MySQLとかPostgreSQLというデータベースを、コード不要でサクッと使えるb>サービス。

Google スプレッドシートのめちゃ高機能版、というイメージ。

div>

📡 なぜ必要？

HTMLファイルだけだとデータを保存できない。

誰かが「募集を投稿」しても、ページを閉じたら消えてしまう。

Supabaseに繋ぐことでみんなの投稿が永続保存される。

STEP 4 — 参加機能の追加
div>

「参加する」ボタンができた理由span>

コミット：「feat: 募集への参加機能を追加 (join_requests)」em>

📋 join_requestsとは？

Supabaseの中に作った「テーブル（表）」の名前。

誰かが「参加する」を押したとき、
誰が・どの募集に・いつ参加したかを
この表に記録する仕組み。

Excelで言えば1枚のシート。

div>

  • JavaScriptが動く
  • Supabaseに「参加データ」を送信
  • 参加者数のカウントが増える
  • Discord IDを表示する

STEP 7 — フィルター & タグ機能
div>

絞り込み検索と雰囲気タグを追加

コミット：「feat: 雰囲気タグ・フィルター・残り時間・Discord IDコピー機能追加」

🏷️ 雰囲気タグとは？

募集に「わいわい」「ガチ」「初心者OK」などラベルを付ける機能。

HTMLの data-* 属性とCSSクラスを使って色分けされたバッジとして表示。

ガチ わいわい 初心者OK
div>
div>

🔍 フィルター機能

「ガチだけ表示したい」など絞り込む機能。

JavaScript が各カードの属性を読んで、条件に合わないカードを非表示b>にする仕組み。

position:sticky でフィルターバーが画面上部に固定される。

div>
div>

📋 Discord IDコピー機能 → ボタンを押すと navigator.clipboard.writeText()code> というJSの命令でコピーされます。

div>
div>
STEP 8 — モックデータ
div>

テスト用のダミーデータ10件を追加

コミット：「feat: モックデータ10件・フィルター sticky・ヒーローパルス追加」

🧪 モックデータとは？

「モック（mock）＝偽物・テスト用」という意味。

実際にSupabaseからデータを取ってくる前に、見た目確認用の仮データをコードに直書きしたもの。

「誰も使ってないのにサイトが空っぽに見える」問題を解決。

div>

✨ ヒーローパルスとは？

CSSアニメーションの一種。

要素が「ふわっ…ふわっ…」と光って点滅するエフェクト。

@keyframes という命令で作り、
「注目させたい箇所」に付ける演出。

div>
div>
/* ヒーローパルス CSS の仕組み */
@keyframes pulse {
  0% { box-shadow: 0 0 0px #ff4655; }
  50% { box-shadow: 0 0 30px #ff4655; }
  100% { box-shadow: 0 0 0px #ff4655; }
}
